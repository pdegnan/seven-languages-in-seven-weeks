# Installing Ruby
See: https://www.ruby-lang.org/en/downloads/
- PC: http://rubyinstaller.org/
- Mac: MacPorts and HomeBrew both have Ruby installers
- Linux - Debian & Ubuntu: sudo apt-get install ruby1.9.1
